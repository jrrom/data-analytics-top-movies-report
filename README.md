# Data Analytics - Top 250 IMDb Movies

Made as a university project for my Data Analytics course. Uses [R](https://www.r-project.org/) and [Tidyverse](https://www.tidyverse.org/).

The main code is in `Case_Study.Rmd`, all output documents are in `docs`.

Build command:

```r
rmarkdown::render("Case_Study.Rmd", output_dir = "output/", output_format = "all")

mv output/Case_Study.html output/index.html
```
Thank you for checking out my report.
